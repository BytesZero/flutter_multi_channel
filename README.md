# weather-flutter

一款用 Flutter 写的天气 App 

### 功能

- 首页（支持下拉刷新）

 1. 首页显示天气和动态背景图
 2. 首页显示今天详细气温数据
 3. 首页显示未来24小时天气（支持横向滑动）

- 一周天气（支持下拉刷新）

 1. 未来7天天气列表（使用``Container decoration``绘制divide）
